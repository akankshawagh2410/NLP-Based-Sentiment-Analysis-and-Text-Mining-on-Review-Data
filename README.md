# NLP-Based-Sentiment-Analysis-and-Text-Mining-on-Review-Data
 
## Overview
 
This project performs **text mining and sentiment analysis** on review data using NLP techniques. It processes textual data to extract meaningful insights about sentiment patterns and word usage.

---
## Key Features
- Tokenization and text preprocessing using NLTK
- Sentiment classification using positive/negative lexicons
- Identification of ambiguous words appearing in both positive and negative contexts
- Extraction of key nouns and patterns from reviews
---
## Tech Stack
- Python
- NLTK
- Pandas
- Regular Expressions (re)
---
## Outcome
- Classified text into sentiment categories
- Identified words with conflicting sentiment usage
- Generated insights on how language varies across reviews
